# 📸 Photo Slideshow Album (Tkinter Project)

A simple and interactive **Photo Slideshow Application** built using **Python, Tkinter, and Pillow (PIL)**.
This project displays images in a slideshow format with a clean graphical user interface.


## 🚀 Features

* 🖼 Displays multiple images in slideshow format
* ▶️ Play button to start slideshow
* ⏱ Automatic 2-second interval between images
* 📏 Image resizing for uniform display
* 🎨 Simple and clean GUI layout


## 🛠 Technologies Used

* **Python**
* **Tkinter** (GUI)
* **Pillow (PIL)** for image handling

## ⚙️ Installation & Setup

1️⃣ Clone the repository:

git clone https://github.com/your-username/Photo-Slideshow-Tkinter.git

2️⃣ Navigate to project folder:

cd Photo-Slideshow-Tkinter

3️⃣ Install required library:

pip install pillow

4️⃣ Run the application:

python Project4.py

## 🖥 How It Works

* Images are loaded using **Pillow**
* Converted into Tkinter-compatible format
* Displayed using a Label widget
* A loop updates the image every 2 seconds
